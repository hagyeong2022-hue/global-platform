# Global Platform — CLAUDE.md

## 한 줄 요약
글로벌 진출 지원 기업들을 한눈에 관리하는 팀 대시보드 — 기업 정보·뉴스·프로그램 현황을 실시간으로.

---

## 기술 스택

| 역할 | 도구 |
|------|------|
| 프론트·서버 | Next.js (App Router, TypeScript) |
| 스타일 | Tailwind CSS |
| 코드 보관 | GitHub |
| 배포 | Vercel |
| AI 기획 | Claude (Plan Mode) |
| 외부 데이터 | Google Sheets API / 네이버 뉴스 API / NICE API / DART API |

> 이 스택 밖의 것은 추가하지 않는다. 필요한 것은 API·외부 도구 수준에서만.

---

## 핵심 원칙

1. **1단계는 일단 돌아가는 최소** — 빈 화면이라도 Vercel URL이 나와야 성공
2. **욕심 금지** — 현재 단계에 없는 기능은 구현하지 않는다
3. **플랜 먼저** — 파일 수정 전에 무엇을 만들지 먼저 정리
4. **데이터 흐름 단순하게** — 구글 시트 → API Route → 화면, 중간 레이어 최소화
5. **Fallback 명확하게** — NICE 실패 시 DART, DART 실패 시 빈칸. 에러로 터지지 않게

---

## 개발 순서 개요

| 단계 | 제목 |
|------|------|
| 1단계 | Next.js 생성 → GitHub 연결 → Vercel 배포 |
| 2단계 | 구글 시트 API 연동 → 기업 데이터 읽기 |
| 3단계 | 기업 목록 테이블 UI + KPI 카드 4개 |
| 4단계 | 기업 상세 페이지 + IR 자료 버튼 |
| 5단계 | 네이버 뉴스 API → 기업별 뉴스 수집 + 메인 뉴스 피드 |
| 6단계 | NICE API + DART fallback → 매출액 자동 수집 |
| 7단계 | 글로벌 프로그램 현황 페이지 + 연도별 바 차트 |
| 8단계 | 검색·필터 + 팀 접근 설정 |
